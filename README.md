# Magic-Task-Board-Drag-and-Drop-Task-Board-Vanilla-JS
Magic Task Board — це SPA-дошка, де можна створювати картки, **перетягувати** їх між колонками *Start / Progress / Done*, задавати **пріоритет** (Low/Med/High), **редагувати/видаляти**, швидко **згортати/розгортати** довгі описи, **експорт/імпорт** даних у JSON та перемикати **темну/світлу тему**. Вся логіка працює повністю в браузері, без сервера.
